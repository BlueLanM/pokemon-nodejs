# 🎮 宝可梦文字版游戏

这是一个基于文字的宝可梦冒险游戏，包含探索、战斗、捕捉、道馆挑战、商店等多个模块。

- [Pokemon](https://bluelanm.github.io/pokemon-nodejs/) - 在线体验

## ✨ 游戏特色

### 🔍 探索模块
- 随机遇到野生宝可梦（使用 PokeAPI）
- 遇到的宝可梦具有随机等级和属性
- 可以选择战斗、捕捉或逃跑

### 🎒 背包系统
- 背包最多可携带 1 只宝可梦
- 显示宝可梦的详细信息（HP、等级、攻击力等）

### 📦 仓库系统
- 当背包满时，捕获的宝可梦自动存入仓库
- 可以查看所有仓库中的宝可梦
- 无限容量

### ⚔️ 战斗系统
- **野外战斗**：可以攻击、捕捉或逃跑
- **道馆战斗**：只能攻击或逃跑
- 回合制战斗，双方轮流攻击
- 战斗胜利后获得金币奖励
- 实时显示战斗日志和 HP 条

### 🎯 捕捉机制
- 捕捉成功率基于精灵球等级
- 捕捉失败会浪费精灵球

### 🏛️ 道馆模块
- 每个道馆有特定的馆主宝可梦
- 战胜道馆获得徽章和丰厚奖励
- 徽章会永久保存

### 🏪 商店模块
- 可以购买各种精灵球
- 显示精灵球价格和捕捉率
- 购买时自动检查金币余额

### 💰 经济系统
- 初始金币：1000
- 战胜野生宝可梦：50-100 金币
- 战胜道馆：500-1000 金币
- 购买物品消耗金币

## 🚀 快速开始

### 环境要求
- Node.js 14+
- MySQL 5.7+

### 后端安装

1. 进入服务器目录：
```bash
cd server
```

2. 安装依赖（包括新增的 axios）：
```bash
npm install
```

3. 配置数据库：
- 复制 `.env.example` 为 `.env`
- 修改数据库连接信息

4. 启动服务器：
```bash
npm run dev
```

服务器会自动创建所有需要的数据表和初始数据。

### 前端安装

1. 进入前端目录：
```bash
cd frontend
```

2. 安装依赖：
```bash
npm install
```

3. 启动开发服务器：
```bash
npm run dev
```

4. 在浏览器打开显示的地址（通常是 http://localhost:5173）

## 🎮 如何游玩

1. **创建角色**
   - 首次进入游戏时输入你的名字
   - 系统会自动给你 1000 金币和 5 个普通球

2. **选择初始宝可梦** ⭐新功能
   - 三选一：小火龙（火系）、妙蛙种子（草系）、杰尼龟（水系）
   - 每只宝可梦都有独特的描述和属性
   - 选择你最喜欢的作为冒险伙伴！

3. **开始探索**
   - 点击"探索"按钮遇到野生宝可梦
   - 使用你的初始宝可梦进行战斗

4. **捕捉宝可梦**
   - 在野外战斗中选择"捕捉"
   - 选择合适的精灵球
   - 捕捉成功后宝可梦会加入背包或仓库

5. **挑战道馆**
   - 点击"道馆"查看所有道馆
   - 选择一个道馆进行挑战
   - 战胜馆主获得徽章和奖励

6. **购买装备**
   - 在商店购买更高级的精灵球
   - 提高捕捉宝可梦的成功率

## 🔄 API 接口

### 玩家相关
- `POST /api/game/player` - 创建玩家
- `GET /api/game/player/:playerId` - 获取玩家信息

### 探索与战斗
- `GET /api/game/explore` - 随机遇到宝可梦
- `POST /api/game/catch` - 捕捉宝可梦
- `POST /api/game/attack` - 攻击

### 道馆
- `GET /api/game/gyms` - 获取所有道馆
- `GET /api/game/gym/:gymId` - 挑战道馆
- `POST /api/game/badge` - 获得徽章

### 商店
- `GET /api/game/shop` - 获取商店物品
- `POST /api/game/shop/buy` - 购买物品

### 背包与仓库
- `GET /api/game/party/:playerId` - 获取背包
- `GET /api/game/storage/:playerId` - 获取仓库
